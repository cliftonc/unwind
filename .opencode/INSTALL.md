# Installing Unwind for OpenCode

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/cliftonc/unwind.git ~/.opencode/plugins/unwind
   ```

2. Configure OpenCode to load skills from this directory.

## Usage

Once installed, Unwind skills will be available for reverse engineering tasks including:
- Codebase analysis
- Service mapping
- API discovery
- Architecture documentation

## Updating

```bash
cd ~/.opencode/plugins/unwind && git pull
```

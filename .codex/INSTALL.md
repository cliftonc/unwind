# Installing Unwind for Codex

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/cliftonc/unwind.git ~/.codex/plugins/unwind
   ```

2. Add to your Codex configuration to load skills from this directory.

## Usage

Once installed, Unwind skills will be available for reverse engineering tasks including:
- Codebase analysis
- Service mapping
- API discovery
- Architecture documentation

## Updating

```bash
cd ~/.codex/plugins/unwind && git pull
```
